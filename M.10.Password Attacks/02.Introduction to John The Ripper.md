# 📌 Introduction to John The Ripper (JtR) - CPTS Notes

---

# 🔍 What is John The Ripper (JtR)

John The Ripper (JtR or john) is a popular password cracking tool used in penetration testing and security auditing.

It supports:
- Dictionary attacks
- Brute-force attacks
- Rule-based attacks
- Cracking password-protected files

The **Jumbo version** is recommended because it includes:
- More hash support
- Better performance
- Extra cracking tools
- File conversion utilities

JtR is widely used because it supports hundreds of hash formats and many attack methods. :contentReference[oaicite:0]{index=0}  

---

# ⚡ Cracking Modes ⭐ IMPORTANT

JtR supports multiple cracking modes depending on the attack scenario.

---

# 1️⃣ Single Crack Mode ⭐ IMPORTANT

## 🔍 Definition
- Rule-based attack mode
- Uses user information to generate password guesses

Uses:
- Username
- Real name
- Home directory
- GECOS values

Applies common modifications:
- Add numbers
- Capitalize letters
- Add symbols

---

## ⭐ MAIN COMMAND

```bash
john --single passwd
```

---

## 📌 Example
Input:
```text
r0lf:$6$ues25dIanlctrWxg$...
```

JtR may generate passwords like:
```text
Rolf123
Sebastian1
r0lf2024
```

---

## 📌 Key Point
Very effective when passwords are based on personal information.

---

# 2️⃣ Wordlist Mode ⭐ VERY IMPORTANT

## 🔍 Definition
- Dictionary attack mode
- Uses password lists instead of random guessing

Most practical attack method in real pentests because users often choose weak passwords.

---

## ⭐ MAIN COMMAND

```bash
john --wordlist=<wordlist_file> <hash_file>
```

---

## 📌 Example

```bash
john --wordlist=rockyou.txt hashes.txt
```

---

# 📚 Popular Wordlists ⭐ IMPORTANT

## rockyou.txt
Contains millions of leaked real passwords.

Location:
```bash
/usr/share/wordlists/rockyou.txt
```

---

## SecLists
Large collection of:
- Password lists
- Usernames
- Fuzzing payloads

---

# 🔧 Rules ⭐ IMPORTANT

Rules modify passwords from wordlists.

Example:
```text
password → Password1
admin → Admin123!
```

---

## ⭐ RULES COMMAND

```bash
john --wordlist=rockyou.txt --rules hashes.txt
```

---

# 3️⃣ Incremental Mode ⭐ IMPORTANT

## 🔍 Definition
- Brute-force style attack
- Generates passwords dynamically
- Uses statistical models (Markov chains)

Does not require a wordlist.

---

## ⭐ MAIN COMMAND

```bash
john --incremental <hash_file>
```

---

## 📌 Key Points
- Most exhaustive attack mode
- Very slow for long passwords
- More efficient than random brute-force

---

# ⚙️ View Incremental Configurations ⭐ IMPORTANT

```bash
grep '# Incremental modes' -A 100 /etc/john/john.conf
```

---

# 🔍 Identifying Hash Formats ⭐ VERY IMPORTANT

Sometimes the hash format is unknown.

Example:
```text
193069ceb0461e1d40d216e32c79c704
```

Use `hashid` to identify possible formats.

---

## ⭐ MAIN COMMAND

```bash
hashid -j <hash>
```

---

## 📌 Example

```bash
hashid -j 193069ceb0461e1d40d216e32c79c704
```

---

## 📌 Output Example

```text
MD5 [JtR Format: raw-md5]
NTLM [JtR Format: nt]
```

---

# ⚙️ Specify Hash Format ⭐ IMPORTANT

If JtR cannot detect the format automatically, specify it manually.

---

## ⭐ MAIN COMMAND

```bash
john --format=<format> <hash_file>
```

---

## 📌 Examples

### NTLM
```bash
john --format=nt hashes.txt
```

### Raw MD5
```bash
john --format=raw-md5 hashes.txt
```

### SHA256
```bash
john --format=raw-sha256 hashes.txt
```

---

# 📂 Cracking Protected Files ⭐ IMPORTANT

JtR supports cracking:
- ZIP archives
- PDFs
- SSH keys
- Office documents
- WPA/WPA2 handshakes

Uses special conversion tools called:
```text
<format>2john
```

---

# ⭐ General Syntax

```bash
<tool> <file> > hashfile
```

---

# 🛠️ Important 2john Tools ⭐ IMPORTANT

| Tool | Purpose |
|------|----------|
| pdf2john | PDF files |
| zip2john | ZIP archives |
| rar2john | RAR archives |
| ssh2john | SSH keys |
| office2john | MS Office docs |
| keepass2john | KeePass DB |
| wpa2john | WPA/WPA2 handshakes |

---

# 📌 Examples

## ZIP File

### Extract Hash
```bash
zip2john file.zip > zip.hash
```

### Crack Password
```bash
john zip.hash
```

---

## PDF File

### Extract Hash
```bash
pdf2john file.pdf > pdf.hash
```

### Crack Password
```bash
john pdf.hash
```

---

# ⚡ CPTS Key Points ⭐

- JtR = multi-purpose password cracking tool
- Wordlist mode = most practical attack
- Incremental mode = brute-force style attack
- Rules improve password guessing
- Use `hashid` for unknown hashes
- Correct hash format improves cracking success

---

# ⚠️ Important Notes

- Incremental mode is resource intensive
- Weak passwords crack quickly
- Hardware affects cracking speed
- Jumbo version is preferred
