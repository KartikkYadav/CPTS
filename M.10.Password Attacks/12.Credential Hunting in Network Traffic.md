# Credential Hunting in Network Traffic

## Overview

Even though modern applications mostly use:
- HTTPS
- TLS encryption

Some environments still use unencrypted protocols such as:
- HTTP
- FTP
- SNMP
- POP3

This allows attackers to capture:
- Usernames
- Passwords
- Hashes
- Community strings

using packet analysis tools like:
- Wireshark
- Pcredz

# Common Protocols

| Unencrypted | Encrypted Version |
|-------------|------------------|
| HTTP | HTTPS |
| FTP | FTPS / SFTP |
| POP3 | POP3S |
| IMAP | IMAPS |
| SMTP | SMTPS |
| LDAP | LDAPS |
| SMB | SMB over TLS |

# Wireshark

## Overview

Wireshark is a packet analyzer used to inspect:
- Live traffic
- PCAP captures

Useful for identifying:
- Credentials
- Sessions
- Cleartext traffic

---

# Useful Wireshark Filters

| Filter | Purpose |
|--------|----------|
| `http` | HTTP traffic |
| `dns` | DNS traffic |
| `tcp.port == 80` | Traffic on port 80 |
| `icmp` | ICMP packets |
| `http.request.method == "POST"` | HTTP POST requests |
| `tcp.stream eq 53` | Specific TCP stream |

---

# Search for Passwords

## Example Filter

```text
http contains "passw"
```

Useful for locating:
- Password fields
- Login forms
- Credentials in POST requests

---

# Important Observation

Unencrypted HTTP POST requests may expose:
- Usernames
- Passwords
- Tokens

directly inside captured traffic.

---

# Pcredz

## Overview

Pcredz extracts credentials from:
- Live traffic
- PCAP files

Supports:
- FTP credentials
- HTTP Basic/NTLM auth
- SMTP/POP/IMAP credentials
- SNMP community strings
- NTLM hashes
- Kerberos hashes

---

# Run Pcredz Against PCAP

## Main Command

```bash
./Pcredz -f demo.pcapng -t -v
```

---

# Example Findings

```text
Found SNMPv2 Community string
FTP User: user
FTP Pass: password
```

---

# Key Points

- Unencrypted traffic may expose credentials
- HTTP POST requests are important targets
- Wireshark filters speed up traffic analysis
- Pcredz automates credential extraction
- Legacy protocols remain dangerous

# One-Line Revision

Credential hunting in network traffic involves analyzing unencrypted protocols with tools like Wireshark and Pcredz to extract usernames, passwords, hashes, and authentication data.
