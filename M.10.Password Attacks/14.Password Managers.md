# Password Managers

## Overview

People use passwords for:
- Wi-Fi
- Social media
- Banking
- Email
- Websites
- Applications

Because users now manage many passwords, they often:
- Reuse passwords
- Choose weak passwords

Password managers solve this problem by securely storing credentials inside an encrypted database.

# What Password Managers Provide

Common features:
- Password storage
- Password generation
- Browser integration
- Auto-fill
- Multi-device sync
- 2FA support
- Security alerts

---

# How Password Managers Work

Most password managers use:
- A master password
- Encryption
- Key derivation functions

The master password is used to:
- Generate encryption keys
- Protect the password vault

Common technologies:
- AES-256 encryption
- PBKDF2-SHA256
- Salted key derivation

## Overview

Cloud password managers:
- Sync passwords across devices

Advantages:
- Convenience
- Cross-device access
- Easy synchronization

# Zero-Knowledge Encryption

Many cloud password managers use:

Meaning:
- Even the provider cannot see stored passwords

# Local Password Managers

## Overview

Local password managers store the database:
- Locally on the device
- Without cloud synchronization

User is responsible for:
- Backups
- Security
- Storage protection

# Alternatives to Passwords

Passwords are not the only authentication method.

Common alternatives:
- MFA
- OTP
- TOTP
- FIDO2
- YubiKey

# Passwordless Authentication

Passwordless authentication removes traditional passwords.

Uses:
- Possession factor
- Biometric factor

Examples:
- Fingerprint
- Security keys
- Authenticator apps

---

# Companies Supporting Passwordless

- Microsoft
- Okta
- Auth0
- Ping Identity

---

# Key Points

- Password managers store credentials securely
- Cloud managers focus on convenience
- Local managers focus on user-controlled security
- Strong encryption protects password vaults
- Passwordless authentication is becoming more common

---

# One-Line Revision

Password managers securely store and manage credentials using encryption, while modern authentication systems are increasingly moving toward passwordless security methods.
