# 🔍 What is Password Cracking
- Process of recovering passwords from hashes
- Hashing = one-way mathematical function
- Common hashes:
  - MD5
  - SHA-256 :contentReference[oaicite:0]{index=0}  

---

# ⚙️ Hash Generation ⭐ IMPORTANT

## MD5 Hash

```bash
echo -n Soccer06! | md5sum
```

Output:
```bash
40291c1d19ee11a7df8495c4cccefdfa
```

---

## SHA-256 Hash

```bash
echo -n Soccer06! | sha256sum
```

Output:
```bash
a025dc6fabb09c2b8bfe23b5944635f9b68433ebd9a1a09453dd4fee00766d93
```

---

# 🌈 Rainbow Tables ⭐ IMPORTANT

## 🔍 Definition
- Precomputed mapping:
  - Password → Hash
- Used for very fast cracking

---

## Example

| Password | MD5 Hash |
|----------|-----------|
| 123456 | e10adc3949ba59abbe56e057f20f883e |
| password | 5f4dcc3b5aa765d61d8327deb882cf99 |
| rockyou | f806fc5a2a0d5ba2471600758452799c |

---

# 🧂 Salting ⭐ VERY IMPORTANT

## 🔍 Definition
- Random bytes added before hashing
- Prevents rainbow table attacks

---

## Example

```bash
echo -n Th1sIsTh3S@lt_Soccer06! | md5sum
```

Output:
```bash
90a10ba83c04e7996bc53373170b5474
```

---

## 📌 Key Points
- Salt is NOT secret
- Usually prepended to hash
- Unique salt per password = best practice

---

# 💥 Brute-Force Attack ⭐ IMPORTANT

## 🔍 Definition
- Tries every possible combination
- Guaranteed success given enough time

---

## 📌 Key Points
- Very slow for strong passwords
- Short passwords (<9 chars) vulnerable
- Speed depends on:
  - Hardware
  - Hash type

---

## ⚡ Example
- MD5 → millions/sec
- DCC2 → thousands/sec

---

# 📖 Dictionary Attack ⭐ VERY IMPORTANT

## 🔍 Definition
- Uses wordlists of common passwords
- Faster than brute-force

---

## ⭐ MAIN COMMAND

```bash
head --lines=20 /usr/share/wordlists/rockyou.txt
```

---

## 📌 Common Wordlists
- rockyou.txt ⭐
- SecLists ⭐
