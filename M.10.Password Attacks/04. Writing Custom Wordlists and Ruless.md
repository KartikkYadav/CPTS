# 🔍 Why Custom Wordlists Matter

Most users create passwords based on:
- Simplicity
- Personal interests
- Predictable patterns

Even when password policies are enforced, users still tend to choose passwords that are:
- Easy to remember
- Related to their life
- Based on company names, pets, hobbies, years, etc. :contentReference[oaicite:0]{index=0}  

Because of this, custom wordlists are often far more effective than generic password lists.

Users commonly modify passwords in predictable ways.

# 🛠️ Hashcat Rules ⭐ VERY IMPORTANT

Hashcat rules modify words from a wordlist to generate more password guesses.

---

# 📌 Common Rule Functions

| Function | Description |
|----------|-------------|
| `:` | Do nothing |
| `l` | Lowercase all letters |
| `u` | Uppercase all letters |
| `c` | Capitalize first letter |
| `sXY` | Replace X with Y |
| `$!` | Add `!` at end |

# ⚙️ Generate Mutated Wordlist ⭐ VERY IMPORTANT

## ⭐ MAIN COMMAND

```bash
hashcat --force password.list -r custom.rule --stdout | sort -u > mut_password.list
```

---

# 📌 Explanation

| Part | Purpose |
|------|----------|
| `-r custom.rule` | Apply rules |
| `--stdout` | Print generated passwords |
| `sort -u` | Remove duplicates |
| `>` | Save output |

# 🌐 Generating Wordlists with CeWL ⭐ VERY IMPORTANT

## 🔍 What is CeWL

CeWL crawls websites and extracts words that can be used in custom password lists.

Useful for:
- Company-specific passwords
- Employee password guessing

---

# ⭐ MAIN COMMAND

```bash
cewl https://www.inlanefreight.com -d 4 -m 6 --lowercase -w inlane.wordlist
```

---

# 📌 Important Options

| Option | Meaning |
|--------|----------|
| `-d 4` | Spider depth |
| `-m 6` | Minimum word length |
| `--lowercase` | Convert words to lowercase |
| `-w` | Output file |

# 📌 Rule Explanation
