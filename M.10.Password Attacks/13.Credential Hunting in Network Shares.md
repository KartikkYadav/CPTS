# Credential Hunting in Network Shares

## Overview

Network shares often contain:
- Passwords
- Config files
- API keys
- Scripts
- Sensitive documents

Common targets:
- IT shares
- HR shares
- Config backups
- Deployment files

Useful keywords:
```text
passw
user
token
secret
cred
key
```

Useful file types:
```text
.ini
.cfg
.env
.xlsx
.ps1
.bat
```

# Snaffler (Windows)

## Overview

Snaffler:
- Finds accessible SMB shares
- Searches for sensitive files
- Looks for credentials automatically

---

## Basic Scan

```cmd
Snaffler.exe -s
```

---

## Useful Options

| Option | Purpose |
|--------|----------|
| `-u` | Search AD usernames |
| `-i` | Include shares |
| `-n` | Exclude shares |

---

## Interesting Findings

Example:
```text
unattend.xml
AdministratorPassword
```

These files may contain:
- Local admin passwords
- Deployment credentials

# PowerHuntShares (Windows)

## Overview

PowerShell-based SMB hunting tool.

Features:
- Enumerates SMB shares
- Finds risky permissions
- Generates HTML report

---

## Basic Scan

```powershell
Invoke-HuntSMBShares -Threads 100 -OutputDirectory c:\Users\Public
```

---

# MANSPIDER (Linux)

## Overview

Linux tool for searching SMB shares remotely.

Best used via Docker.

---

## Search for Passwords

```bash
docker run --rm -v ./manspider:/root/.manspider blacklanternsecurity/manspider 10.129.234.121 -c 'passw' -u 'mendres' -p 'Inlanefreight2025!'
```

# NetExec Spidering

## Overview

NetExec can search SMB shares using:
```text
--spider
```

---

## Main Command

```bash
nxc smb 10.129.234.121 -u mendres -p 'Inlanefreight2025!' --spider IT --content --pattern "passw"
```

---

# What This Does

- Connects to SMB share
- Searches file contents
- Looks for keyword:
```text
passw
```

---

# Key Points

- SMB shares commonly expose credentials
- IT shares are high-value targets
- Search:
  - File names
  - File contents
  - Config files
- Snaffler automates Windows hunting
- MANSPIDER and NetExec useful from Linux

---

# One-Line Revision

Credential hunting in network shares involves searching SMB shares for passwords, secrets, and sensitive files using tools like Snaffler, MANSPIDER, PowerHuntShares, and NetExec.
