# 🌐 Web Enumeration – CPTS Notes

## 🧠 What is Web Enumeration?
- Process of discovering:
  - Hidden files & directories
  - Subdomains
  - Technologies used
- Goal:
  - Expand **attack surface**
  - Find **misconfigurations & vulnerabilities**

---

# 🔍 Directory & File Enumeration

## 🛠 Tool: Gobuster

### 📌 Basic Command
    gobuster dir -u http://<target> -w <wordlist>

### Example
    gobuster dir -u http://10.10.10.121/ -w /usr/share/seclists/Discovery/Web-Content/common.txt

---

## 📊 Important HTTP Status Codes

| Code | Meaning |
|------|--------|
| 200  | Success |
| 301  | Redirect |
| 403  | Forbidden |
| 401  | Unauthorized |

---

## 🎯 Key Findings
- Hidden directories:
  - `/wordpress`
  - `/server-status`
- WordPress detected → **large attack surface**
- Misconfigured setup → possible **RCE**

---

# 🌍 DNS Subdomain Enumeration

## 🛠 Tool: Gobuster (DNS Mode)

### Command
    gobuster dns -d <domain> -w <wordlist>

### Example
    gobuster dns -d inlanefreight.com -w /usr/share/SecLists/Discovery/DNS/namelist.txt

---

## 🎯 Findings
- blog.inlanefreight.com  
- customer.inlanefreight.com  
- my.inlanefreight.com  

👉 Subdomains may expose:
- Admin panels  
- Internal apps  
- Extra attack surface  

---

# 🧰 Web Enumeration Techniques

## 🔹 1. Banner Grabbing (Headers)

### Tool: curl
    curl -IL http://<target>

### Reveals:
- Server type (Apache, Nginx)
- OS (Ubuntu, Linux)
- Frameworks (WordPress, PHP)

---

## 🔹 2. Technology Detection

### Tool: whatweb
    whatweb <target>

### Output includes:
- Web server version  
- CMS (WordPress, etc.)  
- PHP version  

---

## 🔹 3. SSL/TLS Certificates
- Extract:
  - Email addresses  
  - Organization names  
- Useful for:
  - OSINT  
  - Phishing  

---

## 🔹 4. robots.txt

### Example
    http://<target>/robots.txt

### Reveals:
- Hidden directories  
- Admin panels  

👉 Example:
    /private
    /uploaded_files

---

## 🔹 5. Source Code Analysis

### Shortcut:
    CTRL + U

### Look for:
- Credentials  
- API keys  
- Comments  
- Hidden endpoints  

---

# 🔥 Important Tools

- Gobuster → Directory & DNS brute-force  
- ffuf → Advanced fuzzing  
- curl → Header analysis  
- whatweb → Tech fingerprinting  

---

# ⚡ Key Takeaways

## 🧭 Workflow
1. Discover web server (80/443)  
2. Directory brute-force  
3. Subdomain enumeration  
4. Analyze headers & technologies  
5. Check robots.txt  
6. Inspect source code  

---

## 💣 Real-World Insight
- Hidden directories = **entry points**
- WordPress = **high attack surface**
- Misconfigurations → **RCE possibility**

---

# 🏁 Summary
- Web enumeration is **critical phase**
- Always combine:
  - Automation (tools)
  - Manual analysis
- Small findings → can lead to **full compromise**
