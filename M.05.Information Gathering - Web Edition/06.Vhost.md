# 📌 Virtual Hosts (VHosts) & Subdomains

In virtual hosts enum we will find out the subdomains which we missed while doing dns enum in dns enum the subdomain resolve from public dns resolver nut in vhost we resolve that subdomains which are hidden and used in internal network using host file. so for that we need the ip address of the domain we will configure into out /etc/hosts file and will do brute foce .
## 🔍 Key Concept
- Multiple websites hosted on **same IP**
- Differentiated using **HTTP Host header**

---

## ⚖️ Subdomain vs VHost

| Feature      | Subdomain              | VHost                    |
|-------------|----------------------|--------------------------|
| DNS Record  | Required             | Not required             |
| Location    | DNS-based            | Server config            |
| Example     | dev.example.com      | internal.example.com     |

---

## 🧠 Important Points
- VHosts can exist **without DNS records**
- Can access using:
    /etc/hosts (manual mapping)
- Hidden VHosts = **attack surface**

---

# 🔄 How VHost Works

1. Browser sends request  
2. Includes **Host header**  
3. Server checks config  
4. Serves correct website  

👉 Host header = **switch for website selection** :contentReference[oaicite:0]{index=0}

# 📊 Types of Virtual Hosting

## 1. Name-Based
- Uses Host header
- Most common

## 2. IP-Based
- Different IP per site

## 3. Port-Based
- Different ports (80, 8080)

# 🚀 VHost Discovery (Bruteforce)

## 🛠️ Tools
- gobuster ⭐
- ffuf
- feroxbuster

---

# ⭐ MAIN COMMAND (VERY IMPORTANT)

    gobuster vhost -u http://<IP> \
    -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-110000.txt \
    --append-domain

---

## 📌 Example

    gobuster vhost -u http://inlanefreight.htb:81 \
    -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-110000.txt \
    --append-domain

---

## ⚙️ Important Flags
- `-u` → target URL/IP
- `-w` → wordlist
- `--append-domain` → required for correct hostnames
- `-t` → threads (speed)
- `-k` → ignore SSL errors
- `-o` → save output

---

## 📊 Output Example
- forum.inlanefreight.htb → Status 200 ✅
