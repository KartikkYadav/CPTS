# 📌 Web Crawling (Spidering)

## 🔍 Definition
- Automated process to **crawl website pages**
- Extracts:
  - links
  - emails
  - files
  - JS
  - comments

---

## 🎯 Purpose
- Map application structure
- Discover hidden endpoints
- Collect OSINT (emails, files)
- Find attack surface

---

# 🛠️ Popular Crawling Tools

- Burp Suite Spider ⭐
- OWASP ZAP
- Scrapy (Python framework) ⭐
- Apache Nutch

---

# 🚀 Scrapy (Important)

## 📌 Install

    pip3 install scrapy

---

## 📥 Setup ReconSpider

    wget -O ReconSpider.zip https://academy.hackthebox.com/storage/modules/144/ReconSpider.v1.2.zip
    unzip ReconSpider.zip

---

## ⭐ MAIN COMMAND (VERY IMPORTANT)

    python3 ReconSpider.py http://inlanefreight.com

---

## 📊 Output File
- `results.json`

---

## 📌 Extracted Data

- emails → user emails
- links → internal/external URLs
- external_files → PDFs, docs
- js_files → JavaScript files
- form_fields → inputs (login/search)
- images → image paths
- comments → hidden dev notes

---

# ⚡ Why Important (CPTS)

- Emails → phishing / user enum
- JS files → endpoints, API keys
- Comments → hidden info
- Files → sensitive data (PDFs)

---

# 🔥 Attack Flow

1. Crawl website
2. Analyze results.json
3. Extract:
    - endpoints
    - emails
    - hidden files
4. Use for further attacks

---

# ⚠️ Notes
- Can be detected (active recon)
- Avoid high request rate
- Always stay in scope

---
