# DNS Recon Tools (CPTS)

## Overview
DNS reconnaissance uses tools to extract:
- DNS records
- Subdomains
- Infrastructure details

---

## Common DNS Tools

| Tool | Key Features | Use Cases |
|------|-------------|----------|
| `dig` | Advanced DNS queries (A, MX, NS, TXT, etc.), detailed output | Manual queries, troubleshooting, deep analysis |
| `nslookup` | Simple DNS queries (A, AAAA, MX) | Quick domain resolution checks |
| `host` | Fast & concise DNS lookup | Quick A/AAAA/MX lookups |
| `dnsenum` | Automated enumeration, brute-force, zone transfer | Subdomain discovery |
| `fierce` | Recursive search, wildcard detection | Subdomain enumeration |
| `dnsrecon` | Multiple techniques + output formats | Comprehensive DNS recon |
| `theHarvester` | OSINT + DNS + email extraction | Emails, employees, domain info |
| Online Tools | GUI-based DNS lookup | Quick checks without CLI |

---

## Key Notes

- `dig` → Most powerful & flexible  
- `dnsenum / dnsrecon / fierce` → Automation tools  
- `theHarvester` → OSINT-focused (emails + people)  
- `host / nslookup` → Quick & simple  

---

## Summary

- Use combination of tools for better results  
- CLI tools → More control  
- Automated tools → Faster recon  
- OSINT tools → Additional context  

→ Essential for subdomain discovery & attack surface mapping
