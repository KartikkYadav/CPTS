# 🛰️ Introduction to Nmap

## 📌 What is Nmap?
**Network Mapper (Nmap)** is an open-source tool used for:
- Network analysis
- Security auditing

### 🔧 Built With:
- C
- C++
- Python
- Lua

### 🎯 Core Purpose:
Nmap scans networks to:
- Discover active hosts
- Identify open ports
- Detect running services (with versions)
- Identify operating systems
- Analyze firewall / IDS configurations

---

## 🚀 Use Cases

Nmap is widely used by:
- Network administrators
- Security engineers
- Penetration testers

### 🔍 Common Uses:
- Network security auditing
- Penetration testing simulation
- Firewall & IDS validation
- Network mapping
- Response analysis
- Open port identification
- Vulnerability assessment

---

## 🏗️ Nmap Architecture (Core Capabilities)

Nmap scanning is divided into:

1. **Host Discovery**
   - Identify live systems

2. **Port Scanning**
   - Detect open/closed ports

3. **Service Enumeration**
   - Identify services & versions

4. **OS Detection**
   - Detect operating system

5. **Nmap Scripting Engine (NSE)**
   - Automate interaction & vulnerability checks

---

## ⚙️ Basic Syntax

```bash
nmap <scan types> <options> <target>
```

### 🧠 Example:
```bash
nmap -sS 192.168.1.1
```

---

## 🔎 Scan Techniques

```bash
nmap --help
```

### 📡 Common Scan Types:

| Scan Type | Description |
|----------|------------|
| `-sS` | TCP SYN (Stealth Scan) |
| `-sT` | TCP Connect Scan |
| `-sA` | TCP ACK Scan |
| `-sW` | TCP Window Scan |
| `-sM` | TCP Maimon Scan |
| `-sU` | UDP Scan |
| `-sN` | Null Scan |
| `-sF` | FIN Scan |
| `-sX` | Xmas Scan |
| `-sI` | Idle (Zombie) Scan |
| `-sY / -sZ` | SCTP Scans |
| `-sO` | IP Protocol Scan |
| `-b` | FTP Bounce Scan |

---

## ⚡ TCP SYN Scan (-sS)

### 📌 Key Points:
- Default and most popular scan
- Fast (thousands of ports/sec)
- Known as **Stealth Scan**
- Does NOT complete TCP handshake

---

### 🔄 Working:

1. Send **SYN**
2. Analyze response:

| Response | Meaning |
|---------|--------|
| SYN-ACK | Port is **OPEN** |
| RST     | Port is **CLOSED** |
| No Response | Port is **FILTERED** (firewall/IDS) |

---

## 🧪 Example Scan

```bash
sudo nmap -sS localhost
```

---

## 📖 Output Explanation

| Column | Meaning |
|-------|--------|
| PORT | Port number |
| STATE | open / closed / filtered |
| SERVICE | Service running on port |

---

## 🧠 Key Takeaways

- Nmap is essential for **network reconnaissance**
- Helps identify:
  - Open ports
  - Running services
  - System details
- SYN scan is:
  - Fast
  - Stealthy
  - Most commonly used

## 🧩 Pro Tip (Pentesting Perspective)

Combine Nmap with:
- `-sV` → Service version detection  
- `-O` → OS detection  
- `--script` → NSE scripts  

Example:
```bash
nmap -sS -sV -O --script=vuln <target>
```
