# Infiltrating Windows

## Overview

Windows is one of the most targeted operating systems because of:
- Large enterprise usage
- Active Directory integration
- Cloud connectivity
- Huge attack surface

Many major attacks targeted Windows vulnerabilities.

# Important Windows Exploits

| Vulnerability | Description |
|---------------|-------------|
| MS08-067 | SMB vulnerability used by Conficker |
| EternalBlue (MS17-010) | SMBv1 RCE used in WannaCry |
| PrintNightmare | Print Spooler RCE |
| BlueKeep | RDP Remote Code Execution |
| Zerologon | Netlogon authentication flaw |
| SeriousSam | SAM database exposure |
| SigRed | DNS Server RCE |

# Fingerprinting Windows Hosts

## TTL Method

Windows systems commonly reply with:

```text
TTL = 128
```

---

## Ping Example

```bash
ping 192.168.86.39
```

Output:

```text
ttl=128
```

This strongly suggests:
```text
Windows host
```

---

# Nmap OS Detection

## Main Command

```bash
sudo nmap -v -O 192.168.86.39
```

---

# Important Indicators

Example Output:

```text
OS details: Microsoft Windows 10
```

Common Windows ports:
- `135` → MSRPC
- `139` → NetBIOS
- `445` → SMB

# CMD vs PowerShell

## CMD

Best for:
- Older systems
- Simple commands
- Lower logging
- Batch scripts

Prompt:

```text
C:\Windows\system32>
```

---

## PowerShell

Best for:
- Automation
- .NET interaction
- Advanced scripting
- Cloud interaction

Prompt:

```powershell
PS C:\Windows\system32>
```

---

# Important Differences

| CMD | PowerShell |
|-----|-------------|
| Text-based | .NET objects |
| Less logging | More logging |
| Simpler | More powerful |
| Legacy shell | Modern shell |

---

# WSL (Windows Subsystem for Linux)

WSL allows Linux functionality inside Windows.

Attackers may abuse WSL to:
- Execute Linux binaries
- Bypass AV/EDR
- Run Python payloads

---

# Key Points

- Windows commonly identified using TTL and Nmap
- SMB and RDP are common attack targets
- EternalBlue targets SMBv1
- Meterpreter provides advanced shell access
- CMD is stealthier
- PowerShell is more powerful

---

# One-Line Revision

Windows infiltration involves fingerprinting Windows hosts, identifying vulnerable services like SMB, delivering payloads, exploiting vulnerabilities such as EternalBlue, and interacting through CMD or PowerShell shells.
