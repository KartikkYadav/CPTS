# Laudanum - Web Shell Framework

## Overview

Laudanum is a collection of:
- Ready-made web shells
- Reverse shells
- Payload files

Used during:
- Web application exploitation
- File upload attacks
- Remote code execution

It supports multiple web technologies such as:
- ASP
- ASPX
- JSP
- PHP
- ColdFusion

# Laudanum Location

On Kali and Parrot OS:

```bash
/usr/share/laudanum
```

---

# Purpose of Laudanum

Laudanum helps attackers:
- Execute commands through browser
- Obtain reverse shells
- Interact with web servers
- Upload prebuilt payloads quickly

---

# Important Note

Some Laudanum files require modification before use.

Usually:
- Attacker IP must be added
- Callback settings configured

# Common Use Cases

| Attack | Purpose |
|--------|----------|
| File Upload Vulnerability | Upload shell |
| Misconfigured FTP | Direct webroot upload |
| WAR Deployment | JSP shell deployment |
| Weak Validation | Bypass extension checks |

---

# Key Points

- Laudanum contains prebuilt web shells
- Located in `/usr/share/laudanum`
- Supports many web languages
- Upload vulnerabilities commonly lead to RCE
- ASPX shells commonly target Windows IIS servers

---

# Important Notes

- Web shells are often temporary
- Defenders may detect default signatures
- Reverse shells are more stable than browser shells
- Always modify payloads before use

---

# One-Line Revision

Laudanum is a collection of ready-made web shells and payloads used to gain browser-based remote command execution through vulnerable web applications.
