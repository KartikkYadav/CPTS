# ASPX Web Shells & Antak

## Overview

ASPX is a file extension used by:
```text
Microsoft ASP.NET Framework
```

ASPX files run on:
- IIS web servers
- Windows systems

Attackers use ASPX web shells to:
- Execute commands
- Upload/download files
- Gain remote code execution on Windows servers

# How ASPX Works

ASPX pages:
- Process server-side code
- Generate HTML output
- Interact with Windows OS through ASP.NET

Because of this, attackers can upload malicious:
```text
.aspx
```

files to execute commands remotely.

# Antak Web Shell

## Overview

Antak is:
- ASPX web shell
- Part of Nishang framework
- PowerShell-based

Used for:
- Command execution
- File management
- Running PowerShell scripts

---

# Important Modification

Modify:
- Username
- Password

inside shell source code.

Example:

```text
htb-student
```

---

# Why Modify Credentials?

Prevents:
- Random access
- Unauthorized shell usage

Improves:
- Operational security (OPSEC)

---

# OPSEC Recommendations

Before upload:
- Remove comments
- Remove ASCII art
- Obfuscate payload if possible

Reason:
- AV signatures
- Defender detection

# Antak Features

Antak supports:
- PowerShell command execution
- File upload/download
- Encoding commands
- In-memory script execution
- SQL query execution
- Parsing `web.config`

---

# Important Observation

Each command runs:
```text
As a new process
```

through PowerShell.

---

# Useful Antak Functions

| Feature | Purpose |
|---------|----------|

| Upload | Upload files |
| Download | Download files |
| Encode & Execute | Obfuscated execution |
| Parse web.config | Extract config secrets |

# Why Antak is Powerful

Antak is useful because it:
- Uses PowerShell natively
- Works well on Windows IIS servers
- Supports advanced post-exploitation tasks
- Can help deploy reverse shells or C2 agents

---

# Key Points

- ASPX shells target Windows IIS servers
- Antak is part of Nishang
- Uses PowerShell for interaction
- Upload vulnerabilities can lead to full server compromise
- Web shells often provide initial foothold before reverse shell

---

# One-Line Revision

Antak is a PowerShell-based ASPX web shell from Nishang that allows remote command execution and post-exploitation on Windows IIS servers through a browser interface.
